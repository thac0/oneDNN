Creates out-of-place sum_primitive_desc for sum of n inputs multiplied by scale with resulting output_desc memory descriptor.

output_desc can be NULL or specified with the dnnl_format_kind_any format kind – in this case, the appropriate memory format would be chosen automatically.

Inputs:

  • src 0 (dnnl_query_src_md, 0)
  • src 1 (dnnl_query_src_md, 1)
  • ...
  • src n - 1 (dnnl_query_src_md, n - 1)

Outputs:

  • output (dnnl_query_dst_md, 0)
